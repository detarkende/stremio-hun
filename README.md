# stremio-hun

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

To get local HTTPS running:

```bash
mkcert -install -key-file key.pem -cert-file cert.pem localhost
```

This project was created using `bun init` in bun v1.3.11. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
